## Task Format

Each task entry should include:
- **ID**: Short unique identifier (e.g., `TASK-001`)
- **Title**: What needs to be done
- **Context**: Optional 1-2 sentence description
- **Dependencies**: Other tasks that must complete first
- **Assignee**: Agent or human responsible
